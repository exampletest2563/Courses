# Digits Count

## Problem Description

Write a program that prints the count of the digits within an integer number `N`.

## Input

The input consists of a single number - `N`.

## Output

The output consists of a single value - the count of the digits within the number `N`.

## Examples

|Input|Output|
|-|-|
|123|3|
|12423534|8|
|-123|3|
|0|1|
