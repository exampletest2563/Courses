# Descending Sequence

## Problem Description

Write a program that prints all numbers in the interval `[N..1]`.

## Input

The input consists of a single number - `N`.

## Output

The output consists of `N` lines.

## Examples

|Input|Output|
|-|-|
|4|4<br />3<br />2<br />1|
|7|7<br />6<br />5<br />4<br />3<br />2<br />1|
|1|1|
