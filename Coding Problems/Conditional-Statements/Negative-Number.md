# Negative Number

## Problem Description

Write a program that checks whether a given number `N` is negative. The number `N` is negative if it is less than 0.

## Input

The input consists of a single number - `N`.

## Output

The output consists of a single value - `yes` or `no`.

## Examples

|Input|Output|
|-|-|
|1|no|
|-5|yes|
|7|no|
