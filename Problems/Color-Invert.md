# Color Invert

## Problem Description

Write a program that inverts the colors of a given image.

## Input

An image represented as a matrix of color values.

## Output

Display the image with inverted colors.

## Constraints

Your solution must comply with the following constraints:

- The input image is a valid matrix of color values.
